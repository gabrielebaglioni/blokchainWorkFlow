# Infrastructure Scripts

Utility scripts for deployment, maintenance, and operations.

## Scripts

- `setup-local.sh`: Set up local development environment
- `migrate-all.sh`: Run all database migrations
- `deploy.sh`: Deployment script
- `backup.sh`: Database backup script
- `health-check.sh`: Check health of all services

## Usage

```bash
# Make scripts executable
chmod +x infra/scripts/*.sh

# Run setup
./infra/scripts/setup-local.sh
```
