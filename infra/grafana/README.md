# Grafana Configuration

Grafana dashboards and provisioning for monitoring HouseBlock services.

## Dashboards

Pre-configured dashboards for:
- Service health monitoring
- API performance metrics
- Error rates
- Event flow visualization

## Provisioning

Automatic dashboard and datasource provisioning configuration.

## Usage

1. Start Grafana (via Docker or standalone)
2. Import dashboards from `dashboards/`
3. Configure datasources (Supabase, Redis, etc.)
